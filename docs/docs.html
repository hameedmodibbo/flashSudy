Introduction

The Flashcards App is a modern, lightweight, offline-capable tool that helps learners study through spaced repetition and interactive card flipping.

Features

  • Create, flip, and delete cards
  • Dark/Light theme
  • Responsive grid layout
  • LocalStorage saving
  • Study Mode navigation
  • Search & filtering

Usage

Create a card → click card to flip → right-click to delete.

Study Mode

Study cards individually with flip, next, and previous controls.

Responsiveness
